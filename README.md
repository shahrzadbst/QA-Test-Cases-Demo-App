# QA-Test-Cases-Demo-App
Manual QA test cases written for a web demo application to demonstrate structured testing

About this project
This repository contains manual QA test cases written for a publicly available demo web application (the-internet.herokuapp.com). The purpose is to demonstrate structured quality assurance thinking including test case design, expected vs actual results, and defect identification.
Tools used
Manual testing, Google Sheets, GitHub
Test areas covered

Login functionality
Form validation
UI elements and buttons
Error handling
